# Welcome to My Sqlite
***

## Task
The task is to create a Command Line Interface (CLI) for interacting with a SQLite database using Ruby.
The challenge is to allow the user to perform SELECT, INSERT, UPDATE, DELETE, and simple JOIN queries
 interactively, without writing raw SQL.
The CLI also needs to reuse the MySqliteRequest class from my_sqlite_request.rb to build and execute queries.

## Description
we can type an SQL command in the terminal. The CLI parses it and calls the appropriate method in MySqliteRequest.
MySqliteRequest executes the query on the database. The CLI prints the result as an array of hashes

## Installation
put everything in one folder and in the terminal type ruby my_sqlite_cli.rb (look at how_to_test_our_code.text)

## Usage
some tests are ready in test.rb enter ruby test.rb in the terminal. Look also at how_to_test_our_code.text
